Task 1 (read file)

- If `sample.txt` exists, its content will be printed line by line.
- If it does not exist, a friendly error message is shown.

Task 2 (write and append)

- You will be asked to enter text to write to `output.txt`.
- Then you will be asked to enter text to append to the same file.
- Finally, the script prints the complete content of `output.txt`.
